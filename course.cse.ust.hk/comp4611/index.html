Course description

Cloud computing unifies data, users, and programs on a global scale and forms a distributed computing system with unprecedented computing capability. This course studies the design of such systems and related technologies including MapReduce, Spanner, Paxos, Spike and Layer Zero.

The class meets twice a week for a mixture of lectures and class discussions of assigned readings. In addition, students will form groups to develop course projects using a new distributed programming language called c0. We will organize the course projects to be an open-source effort so that students will learn how to create, contribute and collaborate in modern open-source development.

Grading scheme

Grades will be based on class participation, presentation, reading notes, and a course project. Each student will present a few research papers, lead a class discussion, and write notes on some papers in the reading list.

Part of the course work will be supported by the CCMR cloud computing testbed.
