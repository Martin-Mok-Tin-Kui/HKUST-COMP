Logging Out

  • There should be a ‘log out’ button somewhere
  • The ‘button’ could be anything you want – SVG/HTML, etc.
  • All the chat messages are cleared, and the browser returns to the log in page

Automatic Hyperlink

  • There are several ways to add hyperlinks in the chat system
  • One of the ways to implement this requirement is, before a chat message is displayed in the HTML, the content of the message is changed so that every URL becomes a link
  • Then you can use the HTML <a> tag to create a link in the display area
  • You can make use of the indexOf() function to search for an URL inside a chat message
