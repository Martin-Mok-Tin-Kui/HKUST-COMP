Flash example - Random time animation in Flash by setTimeout()

TimelineDescription
  • The setTimeout() function requires two inputs: Syntax to run and time in millisecond
  • Unlike JavaScript, the syntax to run must be a function without brackets and double-quotes.
  • Surprisingly, there is no documentation about this function in Flash Help Window.
ActionScript
in frame 1		 var ran_num = Math.round(Math.random() * 5000)
t_display.text="t = " + ran_num + " ms";
function job(){
    gotoAndPlay(2);
}
setTimeout(job,ran_num);
stop();
