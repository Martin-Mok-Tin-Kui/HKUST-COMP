Flash example - Random time animation in Flash

TimelineDescription
  • This flash example contains four layers, the ActionScript layer has no graph but script code.
  • The first three frames of ActionScript layer contains major ActionScript code. There is no animation in all layers during these three frames.
  • The letter "a" incicates there is ActionScript in that frame.
  • Frames 4 - 50 are responsibles for animation.
ActionScript in frame 1 c_display.text = "Count:0";
var count = 0 ;
var ran_num = Math.floor( Math.random() * 60 ) ;
ActionScript in frame 2 if ( count > ran_num ){
    gotoAndPlay(4);
}
ActionScript in frame 3 count = count + 1 ;
c_display.text = "Count:" + count ;
gotoAndPlay(2);
ActionScript in frame 50 gotoAndPlay(1)
