SVG Space Invaders (Firefox)

An example of using 'hidden' base64 JavaScript code techniques.

To run the game, use the native SVG support of Firefox and click here. Then click View>Source (or Ctrl-U) to see the source code.

(You can be sure you are using the native SVG support, by entering 'about:config' in Firefox, and searching for 'svg', then clicking on the 'svg.enabled' entry until it shows 'true').

The game is a bit slow.
