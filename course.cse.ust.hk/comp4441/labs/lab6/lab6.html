COMP 4441 Computer Music

Lab 6: Filters

What to do?

In this lab, you will import an audio signals from sound file and apply low-pass and high-pass filters to it.

Importing wav file

  • import.orc and import.sco import audio file "808loop.wav".
  • Remember to place the sound file in the same directory as the orchestra and score files.

Lab Assignment - Signal Separation

  • "808loop.wav" is a percussion loop consist of two instrument sounds - bass drum and hi-hat
  • Your task in this lab is to apply appropriate filters to separate the two instrument sounds

 

  • Extract the bass drum from the loop and save your assignment as "drums.orc" and "drums.sco"
  • Apply low-pass filter 2 times with cut-off frequency 100Hz.
  • Remember to balance the signal using the original input.
  • The final results should sound like this

 

  • Extract the hit-hat from the loop and save your assignment as "hi-hat.orc" and "hi-hat.sco"
  • Apply high-pass filter 2 times with cut-off frequency 3000Hz.
  • Remember to balance the signal using the original input.
