COMP 4441 Computer Music

Lab 1: Additive Synthesis in Csound

Setting up the environment of your PC

  • In the start Menu, choose "Creative" => "Creative Audio Control Panel".

  • In the "Speakers" tab, change the "Speaker Configuration" to "Headphones"

Csound Basics

  • Download and save the files, sinewave.orc and sinewave.sco, into your local directory (e.g., C:\Temp).
  • To run Csound in the lab machine, you have to set the correct path for the executables. Download the following batch file to you local directory (e.g., C:\Temp).
  • Double click the SoundPath.bat, a command prompt will pop up and you can start using Csound.
  • Change the working directory to the directory that you saved your sinewave.orc and sinewav.sco files. (Using command "cd")
  • Compile the two files using the following command,

csound sinewave.orc sinewave.sco

  • The compiled sound is immediately played after you press Enter. If you prefer a file output, you have to add an extra flag "-Wo" and add a filename:

csound -Wo sinewave.wav sinewave.orc sinewave.sco

Editing the source code

  • There are various text editor installed in the lab PC. We suggest using "TextPad" since it is intuitive and caused least problems about tabs and encoding

  • In the Start Menu, type "TextPad" and start the program

  • Drag and drop the source files to the program window for editing.

Lab Assignment

  • Make a simple additive synthesis instrument based on the below amplitude/harmonic table. Your final result should sound like this.
  • Please refer to P.7 of the "Additive Synthesis" notes. You may set the duration, attack, and decay to be the same for all notes.
 
Harmonic No Amplitude
1 5311
2 223
3 3886
4 961
5 6972
6 783
7 2487
8 646
9 702
10 1114
11 235
12 2188

Fundamental frequency = 311 Hz

 

Part 1

  • Starting with sinewave.orc and sinewave.sco, modify only the sinwave.sco and save it as additive.sco to accomplish the synthesis.
  • The result can be tested by the following command,

csound sinewave.orc additive.sco -Wo add1.wav

 

Part 2

  • Starting with sinewave.orc and sinewave.sco, modify only the sinwave.orc and save it as additive.orc to accomplish the synthesis.
  • The result can be tested by the following command,

    csound additive.orc sinewave.sco -Wo add2.wav

 

  • Demonstrate your work to your TA after you have finished.
